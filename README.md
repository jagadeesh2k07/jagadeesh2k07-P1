## 📋 Project Overview
This repository contains the source code for my personal responsive portfolio. Moving beyond standard designs, this platform employs a bespoke dual-panel layout: a fixed left branding node paired with a fluid, deep-scrolling right-hand viewport. It serves as an interactive, live-production resume designed to present my technical projects, skill sets, and academic metrics clearly to recruiters and collaborators.

---

## 💡 Key Design & Engineering Features
* **Bespoke Dual-Panel Layout:** Features a fixed, high-contrast left panel for persistent navigation alongside a clean, scrollable content canvas.
* **Ambient UI Elements:** Integrated radiant backdrop effects and smooth transitions to provide a modern, high-fidelity user experience.
* **Synchronized Interactive Telemetry:** Built with a clean typing engine for sub-headers and intuitive indicators to highlight key projects.
* **Fluid Responsive Design:** Uses refined CSS media queries to ensure a seamless transition from complex desktop grid structures to a touch-optimized mobile interface.
* **Defensive Form Handling:** Includes a custom client-side intake form with strict structural verification and validation logic.

---

## 🛠️ Technology Stack & Architecture
* **Structural Blueprint:** HTML5 (Semantic Layout)
* **Design Token System:** CSS3 (Flexbox, Custom Root Variables, Keyframe Animations, and Transitions)
* **Interaction Engine:** JavaScript (ES6+ DOM API, Active Section Scroll Tracking, and Dynamic UI States)
* **Production Deployment:** Vercel Global Edge Network

## ⚙️ Local Deployment & Execution
To clone and run this project locally within a development environment:

Clone the Repository: git clone https://github.com/jagadeesh2k07/Portfolio.git

Configure Environment: Move the project directory into your local XAMPP target folder: C:\xampp\htdocs\portfolio.

Launch: Open your XAMPP Control Panel, start the Apache module, and navigate to: http://localhost/portfolio/
